Syntax

  projected = project(v1, v2, dim)

Input Arguments

  • v1 -- Vector to be projected
  • v2 -- Vector projected onto
  • dim -- [optional] "1" if vectors in row-form (default); "2" if vectors in column form

Output Arguments

  • projected -- Projection of v1 along v2

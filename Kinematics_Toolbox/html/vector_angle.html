vector_angle

Angle between two vectors.

If one of the vectors has zero length, NaN is returned

Contents

  • Syntax
  • Input Arguments
  • Output Arguments

Syntax

  angle = vector_angle(v1, v2, dim)

Input Arguments

  • v1 -- First vector
  • v2 -- Second vector
  • dim -- [optional] index over which the angle is calculated. "1" if vectors in row-form (default); "2" if vectors in column form

Output Arguments

  • angle -- Corresponding angles between the vectors [radians]

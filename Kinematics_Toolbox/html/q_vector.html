q_vector

Extract the vector part from a full quaternion.

Contents

  • Syntax
  • Input Arguments
  • Output Arguments
  • Notes
  • Examples

Syntax

  vector = q_vector(in_quat)

Input Arguments

  • in_quat -- Quaternions or quaternion vectors. (Quaternion vectors are expanded to unit quaternions.)

Output Arguments

  • vector -- Corresponding quaternion vector(s)

Notes

More info under http://en.wikipedia.org/wiki/Quaternion

Examples

   q_vector([cos(0.2), 0, 0, sin(0.2);
             cos(0.1), 0, sin(0.1), 0])
ans =
         0         0    0.1987
         0    0.0998         0
